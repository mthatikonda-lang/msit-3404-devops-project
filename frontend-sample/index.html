MSIT 3404 DevOps Project

Containerized Web Application with Microservices Architecture

Project Team:

Manusha Thatikonda and Karthik K

Project Overview:

This project demonstrates a full-stack DevOps implementation featuring containerized frontend and backend services, showcasing modern cloud-native development practices.

Technology Stack:

  • Frontend: HTML5, CSS3, JavaScript with nginx
  • Backend: Flask (Python) RESTful API
  • Containerization: Docker
  • Orchestration: Kubernetes (Minikube)

Key Features:

  • Microservices architecture with separate frontend and backend
  • RESTful API for inter-service communication
  • Dockerized applications for consistent deployment
  • Kubernetes orchestration for scalability
